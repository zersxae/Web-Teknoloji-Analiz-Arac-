📚 API Dokümantasyonu

Web Teknoloji Analiz Aracı API'sini kullanarak kendi uygulamalarınızı geliştirebilirsiniz

🏠 Ana Sayfaya Dön

GET /api/analyze

Belirtilen URL'deki web sitesinin teknik analizini yapar.

Parametreler

url - Analiz edilecek web sitesinin URL'i (zorunlu)

Örnek İstek

GET /api/analyze?url=https://example.com

Başarılı Yanıt (200 OK)

{
  "url": "https://example.com",
  "backend": {
    "server": "nginx/1.18.0",
    "detected_frameworks": ["Laravel", "PHP"]
  },
  "frontend": {
    "javascript_frameworks": ["Vue.js"],
    "css_frameworks": ["Tailwind CSS"]
  },
  "security": {
    "ssl_score": 90,
    "headers": {
      "Content-Security-Policy": "..."
    }
  }
}

GET /api/extract

Web sitesinin HTML, CSS ve JavaScript kaynak kodlarını çeker ve düzenler.

Parametreler

url - Kaynak kodu çekilecek web sitesinin URL'i (zorunlu)

Örnek İstek

GET /api/extract?url=https://example.com

Başarılı Yanıt (200 OK)

{
  "html": "...",
  "css": {
    "files": [
      {
        "url": "https://example.com/style.css",
        "content": "..."
      }
    ],
    "inline": []
  },
  "javascript": {
    "files": [],
    "inline": ["..."]
  }
}

⚠️ Kullanım Limitleri

  • IP başına dakikada maksimum 60 istek
  • Her analiz için maksimum 30 saniye timeout süresi
  • Maksimum dosya boyutu: 10MB

🚫 Hata Kodları

Kod Açıklama
400 Geçersiz istek (URL parametresi eksik veya hatalı)
429 Çok fazla istek (Rate limit aşıldı)
500 Sunucu hatası
